¿Es necesaria la pantalla de intro en Practice Sounds?

La pantalla actual muestra el símbolo IPA, texto explicativo y dos botones ("Ver lección" / "Practicar ahora"). La pregunta es si este paso intermedio ayuda o frena al usuario.

Contexto: El usuario ya eligió practicar un sonido específico desde la pantalla de Practice Sounds. Llega con intención clara. La pantalla actual es un paso extra entre elegir el sonido y empezar a practicarlo.

Opción A

Directo al ejercicio

Eliminar la pantalla de intro. El usuario entra directo al primer ejercicio. La lección se accede mediante un botón discreto dentro del shell de práctica.

  • La teoría queda enterrada
